# CI/CD Pipeline Migration – GitHub Actions

## Overview
This repository contains automation scripts and workflows built using **GitHub Actions** to enable a CI/CD process for migrating data pipelines from **Mage** to **Databricks**.  
It includes migration utilities, deployment logic, and pipelines successfully promoted to production.

---

## Objectives
- Automate CI/CD using GitHub Actions  
- Migrate pipelines from Mage → Databricks  
- Standardize deployment workflow  
- Ensure reliable production releases  

---

## Key Features
- CI/CD automation via GitHub Actions  
- Pipeline migration scripts  
- Databricks deployment integration  
- Environment-based configurations  
- Production-ready pipelines
